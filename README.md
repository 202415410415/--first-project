**项目介绍**
 
本项目是基于 Java 语言开发的控制台版学生成绩管理系统，采用面向对象思想设计，可完成学生信息与课程成绩的基础管理。
系统结构清晰、代码简洁，适合 JavaWeb 初学者学习与二次开发，需jdk8+,tomcat10+版本框架依赖，运行即可使用。
 
项目用途

该系统主要用于对班级学生信息和各科成绩进行统一管理，支持添加学生、录入成绩、查询成绩、显示所有学生信息等常用操作，能够帮助教师或管理员快速完成简单的成绩统计与信息维护，是学习 Java 基础语法、集合与流程控制的典型实战项目。
 
使用方法
 
1. 将项目源码导入支持 Java 的开发工具（IDEA），配置好 JDK 环境。

2.配置tomcat本地服务器，添加javaWeb依赖。
 
3. 配置完成直接运行程序选择登录角色进入交互式界面，根据控制台提示输入对应数字，即可执行对应的信息管理与成绩操作。
